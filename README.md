# Restaurantliste Backend
**Team-Mitglieder:**

## Projektbeschreibung
Die Restaurantliste-App ist eine smarte Webanwendung, mit der du deine Lieblingsrestaurants aus den besten Metropolen der Welt sammelst und organisierst.  
Nutzer:innen entdecken Top Empfehlungen großer Städte, bauen eigene Favoritenlisten auf und planen kulinarische Abenteuer. 

**Ziel:** Dein persönlicher Gourmet-Kompass für unvergessliche Essenserlebnisse weltweit!

## Hauptfunktionen
- Top Restaurants pro Stadt anzeigen (z.B. Name, Stadt, Küche, Bewertung)
- Restaurants zu einer persönlichen Favoritenliste hinzufügen
- Favoriten in einer übersichtlichen Liste anzeigen
- Restaurants bearbeiten (z.B. Küche oder Bewertung ändern)
- Restaurants aus Favoriten löschen
- Filter nach Stadt, Küche oder Bewertung
- Favoriten nach Bewertung oder Stadt sortieren
